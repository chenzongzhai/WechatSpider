# WechatSpider
实现微信公众号历史文章列表页的批量采集
==================================

* 支持公众号自动翻页
* 支持自动切换公众号

## 依赖

* anyproxy
    * https://github.com/alibaba/anyproxy
* python web.py
    * http://webpy.org/

### 模拟中间人提取微信客户端返回信息
